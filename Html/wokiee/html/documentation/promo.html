Promo settings

* Html block contains a 'tt-promo-box' class. It contains the default settings. By default all the center and positioning of content can be changed with the addition of classes:

  • .tt-point-v-t - vertical top
  • .tt-point-v-b - vertical bottom
  • .tt-point-h-l - horizontal left
  • .tt-point-h-r - horizontal right

** If add class 'hover-type-2' (class="tt-promo-box hover-type-2">) - hover is changing.

*** If add class 'tt-one-child' (class="tt-promo-box tt-one-child">) - change the style for layout (listing-collection.html)
