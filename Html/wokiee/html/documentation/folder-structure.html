Folder StructureL

When you unpack downloaded archive from ThemeForest.net you'll get folder containing 2 folders.

Here they are, sorted alphabetically:

  1. Documentation - Documentation for Wokiee HTML site template
  2. HTML - Main folder for template

HTML folder structure:

  • /HTML
    • /_skins (Template skins) *can be deleted if not used
    • /css (Template CSS and SCSS)
    • /external (Template js)
    • /font (/font/demo.html - demo fonts)
    • /images (All Images)
      • /blog (Blog Images)
      • /custom (Images)
      • /lookbook (Lookbook Images)
      • /portfolio (Portfolio Images)
      • /product (Product Images)
      • /promo (Promo Images)
      • /slides (Slides Images)
    • /js (Template initialization js)
    • /sound (Template sound)
    • /video (Template video)
