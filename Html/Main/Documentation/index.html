Installing The Template

General Settings:

Setting the Navbar logo & Logo title

  1. For setting the logo find the line in HTML file: 
    <a class="navbar-brand" href="#"> <src="http://placehold.it/200x70" alt="logo">
    set your logo in src="logo.png"
  2. For setting logo title: 
    <div class="logo-title">Logo-title </div>

Setting the base color scheme

  1. Locate css/color-schemes folder, you have all the colors in there.
  2. Choose your color.
  3. Open the html files.
  4. Find the line given below and replace the color name with your chosen color.
<link rel="stylesheet" href="css/color-schemes/black.css">

Skill Bar & Pie chart Settings

#1.Skill Bar:

The default state has the skill bar with some value but if you want to set it accoring to your need:

  1. Open the html file.
  2. Find 
    <div class="bar" data-value="70" role="progressbar" >
  3. Set your value in data-value="your value" along with 
    <div class="label numbers"> your value div>
  4. you must put the same value in here the bold area:
<div class="bar" data-value="70" role="progressbar">here</div>

#1.Pie chart:

The default state has the pie chart with some value but if you want to set it accoring to your need:

  1. Open the piechart.js file in js folder
  2. Find in the piechart.js file
    <value:, >
  3. Set your number in value.
  4. you must change the value by checking the circle ID [e.g. id: 'circles-1',] and change according to ID

Fun Factors Settings

The default state has the fun factor with some value but if you want to set it accoring to your need:

  1. Open the html file.
  2. Find 
    <h2 class="timer numbers" data-from="0" data-to="100" >
  3. Set your value in data-to="your value"
  4. For setting background image,find the respective class and path in funfactor sections in styles.css file.

Google Maps

to set the Latitude and Longitude

  1. Open the custom.js file.
  2. Find: 
    var marker = new google.maps.Marker({
                position: new google.maps.LatLng(40.7127, -74.0059),
                map: map,
                title: 'Find us here!'
  3. Type your Latitude and Longitude in 
    new google.maps.LatLng(40.7127, -74.0059)
    use Google Maps to find your Lat, Long.

CSS Structure

We're using two main CSS files styles.css-which is main css, and color-schemes.css css file -for predifined color schemes. we do not recommend making changes to any css files.


We have used different color schemes in different templates.Finished color scheme template is in the /css/color-schemes/ directory:


We have some othes css files for different purpose, some are used to furnish the template,(animate.min.css, responsive.css, etc) and also some are related to respective plugins, (YTPlayer.css, maginific-pop-up.css etc). To customize the template it is Not Necessary to edit these css files.

JavaScript Structure

We're using custom.js as our main Javascript file,which is well organized with necessary information as comented form.


We have used different js files in different pupose.Finished color scheme template is in the /css/color-schemes/ directory:

  • smoothscroll.js - for smooth scrolling
  • solidscript.js & fss.min.js - animated block
  • jquery.countTo.js - countdown effect for fun factor
  • animated-headlines.js - for text rotating
